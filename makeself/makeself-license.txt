  You are about to install hibenchmarks to this system.
  hibenchmarks will be installed at:

                    /opt/hibenchmarks

  The following changes will be made to your system:

  # USERS / GROUPS
  User 'hibenchmarks' and group 'hibenchmarks' will be added, if not present.

  # LOGROTATE
  This file will be installed if logrotate is present.

   - /etc/logrotate.d/hibenchmarks

  # SYSTEM INIT
  This file will be installed if this system runs with systemd:

   - /etc/systemd/system/hibenchmarks.service 

   or, for older Centos, Debian/Ubuntu or OpenRC Gentoo:

   - /etc/init.d/hibenchmarks         will be created

  
  This package can also update a hibenchmarks installation that has been
  created with another version of it.

  Your hibenchmarks configuration will be retained.
  After installation, hibenchmarks will be (re-)started.

  hibenchmarks re-distributes a lot of open source software components.
  Check its full license at:
  https://github.com/firehol/hibenchmarks/blob/master/LICENSE.md
